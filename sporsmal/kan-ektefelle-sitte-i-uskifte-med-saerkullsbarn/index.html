Arv og skifte

Kan ektefelle sitte i uskifte med særkullsbarn?

Ektefelle kan sitte i uskifte med særkullsbarn bare hvis særkullsbarnet samtykker. Uten samtykke må det skiftes med særkullsbarnet.

Kort svar

Ja, men bare med samtykke. Har avdøde særkullsbarn, kan lengstlevende ektefelle ikke overta boet uskiftet med dette barnet uten at særkullsbarnet samtykker etter arveloven. Felles barn trenger ikke samtykke på samme måte.

Hvorfor særkullsbarn står annerledes

Uskifte utsetter arveoppgjøret. For felles barn betyr det at arven normalt venter til lengstlevende dør. Særkullsbarn er ikke barn av lengstlevende, og loven gir dem derfor en sterkere posisjon.

✅ Huskeregel: Felles barn må normalt vente. Særkullsbarn kan kreve oppgjør nå hvis de ikke samtykker til uskifte.

Praktisk eksempel

Ola dør 2. februar 2026. Han var gift med Mari og hadde datteren Frida fra et tidligere forhold. Mari ønsker uskifte. Frida kan samtykke, men hun kan også nekte. Nekter hun, må Mari skifte med Frida før uskifte eventuelt kan gjelde resten.

Hva kan du gjøre nå?

Avklar først hvilke barn som er felles barn og hvilke som er særkullsbarn. Be om skriftlig samtykke hvis uskifte ønskes. Hvis særkullsbarnet nekter, må boet beregne og gjøre opp arven til særkullsbarnet.
