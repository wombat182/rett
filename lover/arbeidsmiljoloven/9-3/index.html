Arbeidsmiljøloven

§ 9-3 — Arbeidsgiver kan bare spørre om nødvendige helseopplysninger

Kan arbeidsgiver spørre om helsen din før ansettelse? Bare helseopplysninger som er nødvendige for arbeidsoppgavene kan kreves.

Kort svar

Arbeidsgiver kan ikke be om helseopplysninger bare fordi de er nysgjerrige. De kan bare spørre om helseopplysninger som er nødvendige for å utføre arbeidsoppgavene i stillingen. De kan heller ikke prøve å skaffe slike opplysninger på andre måter.

Paragraftekst

(1) Arbeidsgiver må ikke i utlysningen etter nye arbeidstakere eller på annen måte be om at søkerne skal gi andre helseopplysninger enn dem som er nødvendige for å utføre de arbeidsoppgaver som knytter seg til stillingen. Arbeidsgiver må heller ikke iverksette tiltak for å innhente helseopplysninger på annen måte. (2) Departementet kan i forskrift gi nærmere bestemmelser om hvilke opplysninger som kan innhentes etter denne paragraf.— Kilde: Lovdata

Hva betyr dette på vanlig norsk?

Denne paragrafen gjelder før ansettelse. Den setter grenser for hva arbeidsgiver kan spørre deg om når du søker jobb.

Hovedregelen er enkel: Arbeidsgiver skal ikke grave i helsen din. Helseopplysninger er private. Det kan handle om sykdom, diagnoser, medisiner, tidligere sykmeldinger, psykisk helse, fysisk funksjon, graviditet, rusbehandling, operasjoner, skader eller andre forhold som sier noe om kroppen eller helsen din.

Arbeidsgiver kan bare be om helseopplysninger som er nødvendige for å utføre arbeidsoppgavene som hører til stillingen. Det betyr at spørsmålet må ha en konkret forbindelse til jobben. Det holder ikke at arbeidsgiver gjerne vil ansette noen som «sjelden er syk». Det holder heller ikke at arbeidsgiver vil unngå fremtidige kostnader eller fravær.

Det viktigste ordet i paragrafen er «nødvendige». Arbeidsgiver må kunne forklare hvorfor opplysningen trengs for akkurat denne stillingen. Et generelt ønske om trygghet er ikke nok. Et spørsmål som kan være lov i én jobb, kan være ulovlig i en annen.

Hvis du søker jobb som pilot, sjåfør, dykker, røykdykker, politi, brannkonstabel eller i en annen stilling med klare sikkerhetskrav, kan helse være relevant. Hvis du søker kontorjobb, butikkjobb eller vanlig kundeservice, skal det mye mer til før arbeidsgiver kan kreve helseopplysninger.

Paragrafen gjelder både stillingsutlysningen og andre måter å spørre på. Arbeidsgiver kan ikke skrive i annonsen at «søkere må opplyse om tidligere sykefravær». De kan heller ikke spørre på intervjuet: «Har du noen sykdommer vi bør vite om?» De kan heller ikke sende et skjema der du skal krysse av for diagnoser.

Arbeidsgiver kan heller ikke innhente helseopplysninger på annen måte. Det betyr at arbeidsgiver ikke kan prøve å omgå regelen ved å kontakte tidligere arbeidsgiver, søke i sosiale medier, spørre bekjente, be om generelle legeopplysninger eller bruke uformelle kanaler. Det er ikke bare direkte spørsmål til deg som er begrenset. Også forsøk på å skaffe helseopplysninger utenom deg rammes.

Dette betyr ikke at helse aldri kan være relevant. Hvis stillingen krever tunge løft, kan arbeidsgiver spørre om du kan utføre arbeidsoppgavene. Hvis jobben innebærer nattarbeid, kan arbeidsgiver forklare arbeidstidsordningen og spørre om du kan jobbe slik. Hvis jobben krever førerkort og sikker kjøring, kan arbeidsgiver stille relevante spørsmål knyttet til det. Men spørsmålene må handle om arbeidsevne og nødvendige krav, ikke privat helsehistorikk.

Det er også forskjell på å spørre om du kan utføre arbeidet, og å spørre om diagnoser. Arbeidsgiver kan ofte spørre: «Kan du utføre arbeidsoppgavene som er beskrevet, inkludert løft på inntil 20 kilo?» Det er noe annet enn: «Har du ryggproblemer?»

Andre ledd sier at departementet kan gi forskrift om hvilke opplysninger som kan innhentes. Det betyr at mer detaljerte regler kan komme i forskrift for bestemte situasjoner. Men hovedregelen i § 9-3 står fast: bare nødvendige helseopplysninger.

Kan arbeidsgiver spørre om sykefravær på jobbintervju?

Som hovedregel nei. Tidligere sykefravær er en helseopplysning. Arbeidsgiver kan ikke be om slike opplysninger bare fordi de vil vite om du har vært mye borte fra jobb.

Et spørsmål som «Hvor mye sykefravær hadde du i forrige jobb?» er derfor normalt problematisk. Det handler ikke direkte om du kan gjøre arbeidsoppgavene i den nye stillingen. Det kan også føre til at søkere med tidligere sykdom blir sortert bort.

Arbeidsgiver kan derimot beskrive jobben konkret og spørre om du kan utføre den. Hvis stillingen innebærer tunge løft, lange vakter, nattarbeid, arbeid i høyden eller ansvar for sikkerhet, kan arbeidsgiver spørre om du kan håndtere disse kravene. Da handler spørsmålet om jobben, ikke om din private journal.

Det er også lov å stille krav som faktisk er nødvendige. Hvis jobben krever helseattest etter lov eller forskrift, blir situasjonen annerledes. Da er det ikke arbeidsgivers nysgjerrighet som styrer, men et rettslig krav. Slike tilfeller henger også sammen med § 9-4 om medisinske undersøkelser.

Hva skjer hvis arbeidsgiver spør ulovlig?

Du kan svare at spørsmålet ikke er relevant for stillingen. Du kan også si at du kan utføre arbeidsoppgavene som er beskrevet, uten å gi private helseopplysninger.

Hvis spørsmålet stilles i intervju, kan det være vanskelig å protestere der og da. Mange er redde for å miste jobbsjansen. Men du kan svare rolig: «Jeg kan utføre arbeidsoppgavene som er beskrevet. Jeg ønsker ikke å gi private helseopplysninger som ikke er nødvendige for stillingen.»

Hvis arbeidsgiver bruker helseopplysninger til å sortere deg bort, kan saken også få side til diskrimineringsreglene. Det gjelder særlig hvis spørsmålet handler om funksjonsnedsettelse, graviditet eller andre vernede forhold. Arbeidsmiljøloven kapittel 13 og likestillings- og diskrimineringsreglene kan da bli relevante.

Eksempeler

Eksempel · Anne

Anne søker jobb som butikkmedarbeider. På intervjuet spør butikksjefen: «Har du vært mye sykmeldt før? Vi trenger folk som ikke blir borte.» Anne trenger ikke svare på dette. Spørsmålet gjelder helseopplysninger som normalt ikke er nødvendige for å utføre arbeidsoppgavene i en vanlig butikkstilling. Arbeidsgiver kan stille spørsmål om arbeidstid, kundebehandling, kassearbeid, varepåfylling og om Anne kan utføre oppgaver som er beskrevet. Men de kan ikke be om en oversikt over sykefraværet hennes. Hvis jobben innebærer tunge løft, kan arbeidsgiver heller spørre konkret: «Stillingen innebærer jevnlig løfting av varer på 15 kilo. Kan du utføre dette arbeidet?» Det er mer relevant og mindre privat.

Eksempel · Thomas

Thomas søker jobb som brannkonstabel. Stillingen innebærer røykdykking, akutt innsats, tungt fysisk arbeid og ansvar for liv og helse. I rekrutteringsprosessen stilles det krav om helse og fysisk skikkethet. Her kan arbeidsgiver ha adgang til å innhente relevante helseopplysninger, fordi de kan være nødvendige for å utføre arbeidsoppgavene. Det betyr ikke at arbeidsgiver kan spørre om alt mulig. Opplysningene må fortsatt være nødvendige og knyttet til stillingen. Men grensen er annerledes enn for en vanlig kontorjobb. Dette viser at § 9-3 ikke gir samme svar i alle jobber. Det avgjørende er hva stillingen krever.

Vanlige feil

  • Spørre jobbsøkere om tidligere sykefravær
  • Bruke helseskjema for alle søkere uansett stilling
  • Be om diagnoser når det holder å spørre om arbeidsevne
  • Kontakte tidligere arbeidsgiver for å få vite om sykefravær
  • Tro at helseopplysninger er greit hvis søkeren «svarer frivillig»
  • Stille samme helsespørsmål til alle stillinger uten konkret behov
  • Blande helsekrav med generelle ønsker om «robuste ansatte»

Hva bør du gjøre?

Les stillingsannonsen nøye. Hvis annonsen ber om helseopplysninger, spør deg selv om opplysningene faktisk er nødvendige for arbeidsoppgavene. Hvis ikke, kan kravet være problematisk.

På intervju bør du skille mellom arbeidsevne og helsehistorikk. Du kan gjerne svare på om du kan utføre konkrete oppgaver. Du trenger normalt ikke gi diagnose, sykmeldingshistorikk eller privat medisinsk informasjon.

Hvis arbeidsgiver ber om helseattest, spør hva grunnlaget er. Er det lov, forskrift eller et konkret sikkerhetskrav? Hvilke opplysninger trengs? Hvem skal se dem? Hvor lenge lagres de?

Ta vare på stillingsannonse, e-poster, skjemaer og meldinger hvis arbeidsgiver ber om helseopplysninger. Hvis det senere blir en sak, er dokumentasjon viktig.

Dumme spørsmål

Kan arbeidsgiver spørre om jeg har vært sykmeldt før?

Som hovedregel nei. Tidligere sykefravær er en helseopplysning og er normalt ikke nødvendig for å vurdere om du kan gjøre jobben.

Kan arbeidsgiver spørre om jeg har en diagnose?

Bare hvis opplysningen er nødvendig for å utføre arbeidsoppgavene. I vanlige jobber er diagnoser som regel private og irrelevante.

Kan arbeidsgiver spørre om jeg kan gjøre tunge løft?

Ja, hvis tunge løft faktisk er en del av jobben. Det er bedre å spørre om du kan utføre oppgaven enn å spørre om ryggproblemer eller sykdom.

Må jeg svare på ulovlige helsespørsmål?

Nei. Du kan svare at du kan utføre arbeidsoppgavene, men ikke ønsker å gi helseopplysninger som ikke er nødvendige for stillingen.

Gjelder dette bare jobbintervju?

Nei. Det gjelder også stillingsannonser, skjemaer, e-poster og andre måter arbeidsgiver prøver å innhente helseopplysninger på.
